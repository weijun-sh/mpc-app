# Introduction
MPC app

# ready requires
```
npm install ethereumjs-tx@1.3 web3 sha256
```
# Setting Up
## Clone The Repository
To get started, launch your terminal and download the latest version of the app.
```
git clone https://github.com/weijun-sh/mpc-app.git
```

## Run
1. run user
```
cd web3/user
node app
```

2. run mpc
```
cd web3/mpc
node app
```
